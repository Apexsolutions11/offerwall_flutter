# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2024-01-XX

### Added
- Initial release of Offerwall Flutter Plugin
- Support for Android and iOS platforms
- WebView integration for displaying the offerwall
- API service for interacting with offerwall endpoints
- Authentication token management
- Persistent credential storage
- Widget for displaying offerwall in Flutter apps
- Example app demonstrating plugin usage
- Comprehensive documentation and README
